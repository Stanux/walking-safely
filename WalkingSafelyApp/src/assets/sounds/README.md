# Sons da Splash Screen

## Arquivo necessário

Adicione um arquivo de som chamado `splash.mp3` nesta pasta.

## Sugestões de som

Para a animação da splash, recomendo um som curto (1-2 segundos) do tipo:

1. **Whoosh/Swoosh** - Som suave de movimento
2. **Chime** - Som de sino elegante
3. **Tech startup sound** - Som moderno de inicialização

## Onde encontrar sons gratuitos

- [Freesound.org](https://freesound.org) - Sons gratuitos com licença Creative Commons
- [Mixkit](https://mixkit.co/free-sound-effects/) - Sons gratuitos para uso comercial
- [Zapsplat](https://www.zapsplat.com) - Biblioteca de sons gratuitos

## Configuração Android

Após adicionar o arquivo `splash.mp3`, copie-o também para:
```
android/app/src/main/res/raw/splash.mp3
```

Execute:
```bash
mkdir -p android/app/src/main/res/raw
cp src/assets/sounds/splash.mp3 android/app/src/main/res/raw/
```

## Formato recomendado

- Formato: MP3
- Duração: 1-2 segundos
- Qualidade: 128kbps ou superior
